# Simple Scroll Game

このプロジェクトは、シンプルな横スクロールアクションゲームです。  
キャラクターAが画面上の一定位置に固定され、横から流れてくる障害物をジャンプで避け続けます。  
**1分間生き残ればゲームクリア** となります。

## ゲームの特徴
- プレイヤーキャラクターは固定位置に配置
- 横から障害物（壁やブロック）が流れてくる
- ジャンプ操作で障害物を避ける
- スコアはなく、1分間の生存で勝利

## 必要な素材
- キャラクター画像（ユーザーが用意）
- 障害物画像（ユーザーが用意）

## 開発メモ
- JavaScript（または他の言語）で実装予定
- 将来的に機能拡張可能
